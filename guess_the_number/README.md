# Guess The Number 🎯

## Description
A simple Python game where the computer generates a random number and the user tries to guess it.

## Features
- Random number generation
- User input handling
- Loop-based guessing
- Feedback hints (Too high / Too low)

## How to Run

```bash
python main.py
```

## Concepts Used
- while loop
- if-elif-else conditions
- random module
- user input
